== Description ==
For organizations that use UTM instead of lat lon. It allows you to map UTM coordinates on the map, and to convert Lat Lon coordinates on the map to UTM
This plugin uses the JavaScript UTM to Lat Lon converter of Charles L. Taylor which can be found here: http://home.hiwaay.net/~taylorc/toolbox/geography/geoutm.html
== Installation ==
1. Copy the entire /utm/ directory into your /plugins/ directory.
2. Activate the plugin.
